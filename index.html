MetaCritic Game Reviews (Up to August 4th, 2020)

Brian Basaldua

*Work in Progress*


Instructions/features:

  • Platform circle size based on number of game reviews it has. Game circles(after zooming in)size based on MetaCritic Score
  • So far i have two filters, All games and By year. By year contains circles for each year. The size of these are proportional to the number of games published that year.
  • Custom transitions for Consoles that have ALOT of games, reduces lag of zoom animations.
  • Click on circle to zoom in. If on mobile use zoom out button to zoom out. Otherwise click SAME circle to zoom out.
  • Click on little circles to show game info in top right

Javascript used for data visualization(D3) and Python used for webscraping data(BeautifulSoup)

Libraries used/main methods used:

  • D3 library - D3 Pack used to create a Bubble Pack Layout using data tree
  • D3 library - D3 Nest used to nest scraped data
  • D3 library - D3 Heirarchy used to turn nested data into a tree
  • BeautifulSoup library - Used to scrape all game reviews on MetaCritic Website which is then added to a csv
